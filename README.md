# 🎙️ AI Notetaker

An end-to-end AI application that converts audio into clean, structured notes. Record or upload an audio clip and get back a full transcription plus concise bullet-point notes in seconds.

**🔗 Live Demo:** [Hugging Face Space](https://huggingface.co/spaces/saiteja2521/ai-notetaker)

---

## ✨ Features

- 🎤 **Flexible input** — upload an audio file or record directly from your microphone
- 📝 **Accurate transcription** — powered by OpenAI Whisper
- 🧠 **Smart summarization** — concise, bullet-point notes generated by Meta LLaMA 3.1-8B-Instruct
- 🖥️ **Simple UI** — clean Gradio interface, no setup required for end users
- ☁️ **Deployed live** — hosted on Hugging Face Spaces

---

## 🏗️ How It Works

```
Audio (upload / mic)
        │
        ▼
┌──────────────────────┐
│  OpenAI Whisper API  │  →  Transcription
└──────────────────────┘
        │
        ▼
┌──────────────────────────────┐
│  LLaMA 3.1-8B-Instruct        │  →  Bullet-point notes
│  (Hugging Face Transformers)  │
└──────────────────────────────┘
        │
        ▼
   Gradio Interface
```

1. The user provides audio through the Gradio interface.
2. The audio is sent to **OpenAI Whisper** for speech-to-text transcription.
3. The transcription is passed to **LLaMA 3.1-8B-Instruct** with a note-taking prompt.
4. The transcription and generated notes are displayed back to the user.

| Component | Technology |

## 🚀 Getting Started

### Prerequisites

- Python 3.9+
- An [OpenAI API key](https://platform.openai.com/api-keys)
- A [Hugging Face access token](https://huggingface.co/settings/tokens) with access to `meta-llama/Llama-3.1-8B-Instruct` (you must [request access](https://huggingface.co/meta-llama/Llama-3.1-8B-Instruct) to the gated model)
- A GPU is recommended for running the LLaMA model locally

### Installation

```bash
# Clone the repository
git clone https://github.com/<your-username>/ai-notetaker.git
cd ai-notetaker

# (Optional) create a virtual environment
python -m venv venv
source venv/bin/activate        # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt
```

### Configuration

Create a `.env` file in the project root (use `.env.example` as a template):

```env
OPENAI_API_KEY=your_openai_api_key_here
HUGGINGFACE_API_KEY=your_huggingface_token_here
```

### Run

```bash
python app.py
```

Gradio will print a local URL (e.g. `http://127.0.0.1:7860`). Open it in your browser.

## ⚠️ Notes & Limitations

- **Compute:** The public demo runs on Hugging Face's free CPU tier, so responses may be slower than a GPU-backed deployment.
- **API quota:** Transcription depends on the OpenAI Whisper API. If transcription fails, it may be due to API quota limits.
- **Output length:** Note generation is currently capped at a small token limit for speed; increase `max_new_tokens` in `app.py` for longer notes.
